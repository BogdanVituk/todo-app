# Todo Frontend

Сучасний SPA додаток на React 19, оптимізований для швидкої взаємодії та зручного інтерфейсу.

## 🛠️ Технології
- **UI:** React 19 + TypeScript.
- **Форми:** React Hook Form + Zod.
- **API Клієнт:** Axios з налаштованими інтерцепторами.
- **Повідомлення:** React Hot Toast.

## 🧩 Декомпозиція та Хуки
Проєкт використовує кастомні хуки для відокремлення UI від логіки:
- `useFetching`: Універсальна обгортка для обробки Loading та Error станів.
- `useFilteredTodos`: Оптимізована фільтрація та пошук через `useMemo`.

## 🎨 Дизайн
- **Tailwind CSS:** Забезпечує адаптивну верстку та швидку стилізацію.
- **Mobile First:** Інтерфейс коректно відображається на будь-яких пристроях.
- **Interactive UI:** Кнопки мають стани завантаження (`isSubmitting`), а помилки підсвічуються в реальному часі.

### 🔑 Налаштування змінних оточення

Для коректної роботи проєкту необхідно створити `.env` файли:

**Frontend (`/frontend/.env`):**
`VITE_API_URL="http://localhost:3000"`
